# mkdir

**建立目录** 
语　　法：mkdir [-p][--help][--version][-m <目录属性>][目录名称]

* -p : 创建目录的时候如果上层补录不存在会创建上层目录
* -m<目录属性> : 创建目录的同时设置权限 

补充说明：mkdir可建立目录并同时设置目录的权限。
